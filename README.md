# awsm Raycast Extension

Raycast integration for [awsm](https://github.com/AleG03/awsm), the AWS Manager that simplifies switching between AWS profiles, handling MFA, assuming roles, and generating console sign-in links.

## Prerequisites

This extension requires the [awsm](https://github.com/AleG03/awsm) binary to be installed on your system and available in your `PATH`.

## Features

- **Show AWS Profiles**: List all configured AWS profiles.
- **Set Active Profile**: Quickly switch between profiles.
- **Open Console**: Generate a console sign-in link and open it in your browser.
